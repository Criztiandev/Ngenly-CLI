# Ngenly CLI

A node project boilerplate generator

## Installation

Install the package globally using yarn or npm

```console
npm install -g ngenly
```

## Usage

- execute in terminal
  ngenly [command]

### Commands

- Create a new project

```console
ngenly new
```

- Generate CRUD files
```console
ngenly -g crud [expressjs | expressts] [model_name]
```

- View the cli version

```console
ngenly -v
```
